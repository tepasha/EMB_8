Енкодер:
  CLK (A) → GPIO6
  DT  (B) → GPIO7
  SW  BTN → GPIO8
  VCC     → 3.3V
  GND     → GND

Сервопривід:
  сигнал  → GPIO4
  5V      → зовнішнє живлення
  GND     → спільний GND

Зумер:
  + → GPIO5
  − → GND

Три режими — одна константа:
g_mode = MODE_SERVO;   // керування сервоприводом
g_mode = MODE_SAFE;    // сейф (код 3-7-1-5)
g_mode = MODE_CALC;    // калькулятор

Приклад виводу сейфу:
[SAFE] > 3714    ← \b перемальовує останню цифру
[SAFE] Невірно! Спроба 1/3

[SAFE] > 3715
[SAFE] ВІДЧИНЕНО!
Приклад виводу калькулятора:
[CALC] 234 + 77 = 311
[CALC] 0          ← новий вираз
